⚙️ Hardware Installation 

1️⃣ Remove Bottom Case:
Remove 10 Phillips screws, 
Lift bottom cover

2️⃣ Disconnect Battery: 
Disconnect battery from logic board (critical step)

3️⃣ Install RAM:
Remove old RAM,
Insert new RAM at an angle,
Lock clips into place,

4️⃣ Install SSD:
Remove HDD bracket,
Disconnect SATA cable,
Transfer mounting screws,
Install SSD

5️⃣ Replace Battery:
Remove tri-wing screws,
Swap battery,
Reconnect cable,

6️⃣ Reassemble Device:
Reinstall bottom cover,
Secure screws
